EVENTOS EN JAVASCRIPT

Evento

Un evento es una acción o ocurrencia que sucede en el navegador y que puede ser detectada y manejada por el código JavaScript.

getElementById

Es un método que se utiliza para obtener una referencia a un elemento HTML en el documento basándose en su atributo id.

let miElemento = document.getElementById('miElemento');

miElemento.innerHTML = '¡Hola, mundo!';

addEventListener

se utiliza para adjuntar un manejador de eventos a un elemento HTML. Permite que se ejecute una función (o "manejador de eventos") cuando ocurre un evento específico en ese elemento.

EVENT BUBBLING

Es un concepto en JavaScript relacionado con la propagación de eventos a través de la jerarquía de elementos del DOM. Cuando ocurre un evento en un elemento específico, el evento se propaga desde ese elemento hacia arriba a través de sus ancestros en la jerarquía del DOM.

DELEGACIÓN

Aprovecha el concepto de event bubbling para gestionar eventos eficientemente. En lugar de agregar manejadores de eventos directamente a los elementos individuales, puedes asignar un único manejador de eventos a un ancestro común de los elementos y luego determinar el objetivo real del evento dentro de ese manejador.
