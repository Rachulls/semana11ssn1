## EVENTOS
- Un evento es una acción u ocurrencia que sucede en el navegador y puede ser detectada y manejada por el código JavaScript.

## GETELEMENTBYID
- Se utiliza para obtener una referencia a un elemento HTML en el documento basándose en su atributo id.

## ADDEVENTLISTENER
- Se utiliza para adjuntar un manejador de eventos a un elemento HTML.

## EVENT BUBBLING
- Concepto en JavaScript relacionado con la propagación de eventos.

## DELEGACIÓN
- Técnica que aprovecha el concepto de event bubbling para gestionar eventos eficientemente.
